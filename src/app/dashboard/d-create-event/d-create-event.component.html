Maak hier een nieuw evenement aan!

Om te beginnen met het maken van evenementen is het belangrijk om een Stripe account aan te maken. Stripe is ons payout-platform, wat al je geldzaken regelt. Bovendien kan je hier precies zien hoeveel tickets je hebt verkocht en hoeveel je daar mee hebt verdient. Als klanten hun geld terug willen, kan je dat zelf regelen via het Stripe Dashboard. Als je een account hebt aangemaakt, wordt je weer doorgestuurd naar deze website.
