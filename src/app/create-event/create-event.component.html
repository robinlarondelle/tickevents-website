Maak een evenement aan

Via TickEvents kan je makkelijk je evenement hosten. Wij verzorgen de ticketverkoop, jij verzorgt het evenement. Om te beginnen maak je op TickEvents een evenement aan. Dit doe je door simpelweg dit formulier in te vullen. Als je het evenement hebt aangemaakt, kun je het hierna bekijken in je persoonlijke beheerderspaneel, onder 'Mijn Evenementen'.
